RIVALS TPC - 서비스 규칙

📄 이용약관

이 시스템은 데이터 처리 및 정보 제공 서비스입니다.

1. 목적

사용자 요청에 따라 검색 기반 데이터를 제공합니다.

2. 책임

표시되는 모든 정보는 참고용이며 정확성을 보장하지 않습니다.

3. 금지 행위

서버 과부하 유발, 봇 악용, 비정상적인 접근은 금지됩니다.

4. 서비스 변경

운영자는 사전 공지 없이 서비스를 수정 또는 종료할 수 있습니다.

5. 문의

문의는 https://discord.gg/Y9CUtusqXW 여기로 해주세요.

🔒 개인정보 처리방침

최소한의 데이터만 수집됩니다.

1. 수집 정보

  • 사용자 ID
  • 검색 키워드

2. 사용 목적

서비스 기능 제공을 위해서만 사용됩니다.

3. 저장 여부

개인 정보는 장기 저장되지 않습니다.

4. 제3자 제공

수집된 정보는 외부에 공유되지 않습니다.

5. 보안

데이터는 안전하게 보호됩니다.
