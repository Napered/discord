📄 이용약관

본 웹사이트는 정보 제공 및 데이터 조회 기능을 목적으로 운영됩니다.

1. 목적

본 서비스는 사용자 요청에 따라 데이터를 검색하고 표시하는 기능을 제공합니다.

2. 책임

본 서비스에 표시되는 정보는 참고용이며, 정확성이나 완전성을 보장하지 않습니다.

3. 금지 사항

사용자는 본 서비스를 악용하거나 서버에 과도한 부하를 주는 행위를 해서는 안 됩니다.

4. 서비스 변경

운영자는 사전 공지 없이 서비스 내용을 변경하거나 중단할 수 있습니다.

5. 문의

문의 사항은 봇 설명에 디스코드 링크로 들어와 문의방에 문의해 주십시오.

🔒 개인정보 처리방침

본 서비스는 최소한의 정보만을 사용합니다.

1. 수집하는 정보

  • 사용자 ID
  • 입력된 검색 키워드

2. 사용 목적

수집된 정보는 데이터 조회 및 기능 제공을 위해서만 사용됩니다.

3. 저장 여부

본 서비스는 개인정보를 장기적으로 저장하지 않습니다.

4. 제3자 제공

수집된 정보는 외부에 제공되지 않습니다.

5. 보안

데이터는 안전하게 관리됩니다.
